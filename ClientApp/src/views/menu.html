Generar | Scripts backup
  • Genera los scripts necesarios para realizar el backup de información de una tabla.
  • Genera los scripts para CERT y PROD en un repositorio en bitbucket.
  • Esta acción usa las credenciales del datalake.
Generar | Repositorio
  • Elije la plantilla (DevSecops) a utilizar.
  • Crea el repositorio y las ramas: master, develop, construccion en caso sea necesario.
  • Es necesario tener el rol para crear repositorios.
Generar | Jobs Jenkins
  • Se genera jobs pipelines para los ambientes DEV, CERT y PROD.
  • Estos jobs son creados a partir de unos jobs plantillas (appsettings.json).
  • La actualización del url de bitbucket es manualmente posterior a la creación.
